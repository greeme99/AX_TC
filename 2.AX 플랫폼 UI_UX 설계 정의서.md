# **🎨 AX Control Tower UI/UX 설계 정의서**

이 문서는 사용자가 플랫폼을 쉽고 즐겁게 이용할 수 있도록 돕는 시각적 가이드라인입니다.

## **🎨 0\. 전체 디자인 원칙 (Design System)**

* **톤앤매너:** 신뢰감을 주는 'Deep Blue'와 혁신을 상징하는 'Electric Purple'을 포인트 컬러로 사용합니다.  
* **컴포넌트 특징:** 모든 버튼과 카드 형태는 끝을 둥글게(Rounded) 처리하여 현대적이고 부드러운 느낌을 줍니다.  
* **반응형 레이아웃:** PC 대시보드 형태를 기본으로 하되, 모바일에서도 중요한 알림과 과제 현황을 확인할 수 있도록 유동적인 그리드를 사용합니다.

## **🏠 1\. 메인 대시보드 (Home)**

"한눈에 보는 우리 회사의 AI 온도"

* **주요 구성 요소:**  
  * **웰컴 배너:** "안녕 하세요, \[사용자명\]님\! 오늘도 AI와 함께 한 걸음 더 성장해 볼까요?" (친근한 메시지)  
  * **전사 AI 현황판:** 현재 진행 중인 과제 수, 절감된 업무 시간(Total ROI), 이번 달 가장 많이 쓰인 AI 모델 TOP 3\.  
  * **바로가기 카드:** 보안 서약(미완료 시 Red 알림), 추천 프롬프트 템플릿, 오늘 마감인 교육 자료.  
* **UX 포인트:** 데이터가 너무 딱딱하지 않게 원형 그래프(Donut Chart)나 부드러운 애니메이션 바(Progress Bar)를 활용합니다.

## **🛡️ 2\. 보안 관리 화면 (Security & Compliance)**

"안전한 AI 활용의 시작점"

* **화면 구성:**  
  * **보안 서약 섹션:** 상단에 크게 '서약 완료 여부' 표시. 서약하기 버튼 클릭 시 모달(팝업) 형태로 서약서 등장.  
  * **가이드라인 라이브러리:** 카드 뉴스 형태로 제작된 '직무별 보안 Do & Don't'.  
  * **프롬프트 자가 진단기:** 입력창에 프롬프트를 넣고 '체크'를 누르면, 위험 단어(고객정보 등)를 하이라이트 해주고 순화 제안.  
  * 코드 자가 진단기 : 입력창에 코드를 넣거나 프로그램이 들어 있는 폴더를 선택하면 분석해서  1\. 하드코딩, 2.입력값 검증 누락, 3.불충분한 권한 통제 4\. 무분별한 데이터 공유 항목 여부 점검=  
* **UX 포인트:** 보안은 '감시'가 아니라 '보호'라는 느낌이 들도록 친절하고 안내적인 문구를 사용합니다.

## **💡 3\. Template 관리 화면 (Prompt Hub)**

"최고의 성과를 내는 AI 레시피 북"

* **화면 구성:**  
  * 프롬프트 Type 필터 : System Prompt, Instruction Prompt, User Prompt  
  * Type별 대표 모델/툴 :   
    1\) System :  Gemini, ChatGpt, Claude, Grok, Perplexity 등  
    2\) Instruction : GEMs, GPTs, Project (GPT, Grok, Claude 등), Skill (Cladue), 스튜디오 (노트북LM)  
    3\) User : LLM 대화창의 Prompt, Context 등

  * **카테고리 필터:** 마케팅, 인사, 개발, 일반행정 등 탭으로 구분.  
  * **템플릿 카드 리스트:** 각 카드에는 템플릿 제목, 설명, 사용 모델(Gemini, ChatGPT 등), 추천 점수(별점) 표시.  
  * **상세 보기 & 복사:** 카드를 클릭하면 전체 프롬프트가 보이고, '한 번에 복사하기' 버튼 제공.  
* **UX 포인트:** '내 프롬프트 저장하기' 기능을 두어 개인화된 경험을 제공합니다.

## **📚 4\. 교육 자료 화면 (Learning Center)**

"AI 마스터로 가는 성장 사다리"

* **화면 구성:**  
  * **학습 로드맵:** 게임의 '스킬 트리'처럼 초급/중급/고급 과정이 선으로 이어져 있어 성취감 부여.  
  * **콘텐츠 플레이어:** 영상 재생창 옆에 강의 요약본(텍스트)과 관련 프롬프트 예시를 함께 배치.  
  * **학습 이력:** "내가 수강한 강의 %" 표시.  
* **UX 포인트:** '완강' 시 배지를 부여하여 지속적인 학습 동기를 유발합니다.

## **📊 5\. AX 과제 관리 화면 (Task Tracker)**

"도전하는 모든 과제는 기록됩니다"

* **화면 구성:**  
  * **칸반 보드 (Kanban):** \[아이디어\] \-\> \[개발 중\] \-\> \[테스트\] \-\> \[완료\] 칸을 이동하는 과제 카드들.  
  * **과제 등록 폼:** 제목, 예상 효과, 담당자, 소요 예산 등을 입력하는 간편한 입력창.  
  * **성과 요약창:** 개별 과제 클릭 시 해당 과제가 창출한 가치를 인포그래픽으로 보여줌.  
* **UX 포인트:** 과제 카드를 드래그 앤 드롭(Drag & Drop)으로 옮길 수 있게 하여 조작의 재미를 더합니다.

## **💎 6\. AI 리소스 관리 화면 (Resource Center)**

"투명하고 합리적인 도구 활용"

* **화면 구성:**  
  * **모델 현황판:** 회사에서 지원하는 모델 목록과 현재 내 계정의 남은 쿼터(Quota) 표시.  
  * **유료 모델 신청 버튼:** 신규 모델이 필요할 때 사유를 적어 제출하는 플로팅 버튼.  
* **UX 포인트:** "신청이 승인되었습니다\!"와 같은 알림을 상단 알림 바에 즉각적으로 노출합니다.

## **🏆 7\. 베스트 과제 DB (Hall of Fame)**

"모두의 박수를 받는 성공의 기록"

* **화면 구성:**  
  * **명예의 전당 배너:** 분기별 가장 뛰어난 ROI를 낸 팀의 사진과 성과 게시.  
  * **검색 & 태그:** 'Python', '자동화', '시간절약' 등 태그로 필요한 성공 사례 검색.  
  * **자료 다운로드:** 해당 과제에서 사용한 매뉴얼이나 코드 샘플(Zip) 다운로드 섹션.  
  * 자료 업로드 : PC 업로드, Drive (Google drive, MS Onedrive 등) 선택하여 업로드   
* **UX 포인트:** 인스타그램처럼 '좋아요'와 '댓글' 기능을 넣어 활발한 사내 공유 문화를 조성합니다.

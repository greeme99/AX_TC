# **🛰️ AX Control Tower: AI 전환(AX) 통합 플랫폼 마스터 플랜**

이 마스터 플랜은 기업의 AI Transformation(AX)을 가속화하기 위한 전략, 설계, 보안 및 개발 지침을 모두 통합한 최상위 문서입니다.

## **🌟 1\. 프로젝트 개요 (Project Vision)**

* **미션:** 사용자를 단순한 코더에서 '코드로 생각하는 사람'으로 변화시키며, 기업 내 AI 활용을 표준화하고 성과를 가산화함.  
* **슬로건:** "AI, 아는 것을 넘어 실행하는 문화로"  
* **성장 사다리:** Spark(아이디어) → Prototype(기능 구현) → Upgrade(고도화) → Identity(자산화)

## **📋 2\. 요구기능 정의 (Functional Requirements)**

### **\[6대 핵심 메뉴\]**

1. **🛡️ 보안관리:** 보안지침서, 전자서약서, 보안점검 툴, 주체별 R\&R 관리.  
2. **💡 Template 관리:** System/Instruction/User 프롬프트 및 레퍼런스 허브.  
3. **📚 교육자료:** 동영상/문서 DB, 온라인 자료(URL) 연동, 직무별 학습 로드맵.  
4. **📊 AX 과제관리:** 과제 등록, 진척 Status(Kanban), ROI 성과 검증 대시보드.  
5. **💎 AI 리소스 관리:** 사용 모델 현황, 유료 모델 신청/승인 프로세스, 사용자 인증 관리.  
6. **🏆 베스트 과제 DB:** 소스코드/MD 파일 아카이브, 명예의 전당, 데이터 업로드(Drive 연동).

## **🎨 3\. UI/UX 화면 정의 (Screen & Experience)**

### **\[디자인 원칙\]**

* **톤앤매너:** Deep Blue (\#1e293b) & Electric Purple (\#7c3aed)의 전문적이고 혁신적인 느낌.  
* **스타일:** Rounded-2xl, Glassmorphism, 유동적 그리드 레이아웃.

### **\[주요 화면 상세\]**

* **메인 대시보드:** 전사 AI 온도계(현황판), 웰컴 배너, 개인별 바로가기 카드.  
* **보안/코드 자가진단:** 프롬프트 및 소스코드 내 민감정보(하드코딩, 권한통제 등)를 AI가 즉시 분석.  
* **프롬프트 허브:** 필터링 기능이 강화된 카드 리스트와 '내 프롬프트 저장하기' 개인화 기능.  
* **과제 트래커:** 드래그 앤 드롭 방식의 칸반 보드 및 인포그래픽 성과 보고서.

## **🛡️ 4\. 개발 및 보안 명세 (Development & Security)**

### **\[핵심 보안 5대 원칙\]**

1. **기밀 최소화:** 개인정보·영업비밀 입력 절대 금지.  
2. **인증 분리:** API Key 등 인증 정보는 소스코드 내 기재 금지 (.env 관리).  
3. **최소 권한:** AI 생성 코드에는 업무에 필요한 최소 권한만 부여.  
4. **검증 후 배포:** 생성 코드는 반드시 사람이 리뷰 후 운영 환경 적용.  
5. **사용 이력 관리:** 입력 프롬프트 포함 모든 로그를 팀 단위로 기록·보관.

### **\[개발 세부 지침\]**

* **Mailing:** SMTP 계정 정보 환경변수 처리, 수신자 목록 외부 참조.  
* **Database:** 실제 데이터 입력 금지(더미 데이터 사용), 테이블명 가명화(SQL 생성 시).  
* **API/파일:** 확장자 필터 적용, 외부 전송 시 관리자 승인 로직 필수.

## **🛠️ 5\. 기술 스택 및 개발 환경 (Tech Stack)**

* **Frontend:** React 18+ (Vite), Tailwind CSS, Lucide React, Recharts.  
* **Backend/DB:** Google Firebase (Auth, Firestore, Storage, Hosting).  
* **IDE:** Google Project IDX (Antigravity Cloud 환경).  
* **AI Integration:** Google Gemini API (Main Model), Serverless Functions (Proxy).

## **🤖 6\. 프로토타입 생성용 마스터 프롬프트 (Generation Prompt)**

### **\[AI 개발 지시서\]**

* **역할:**  기업용 AX 플랫폼 전문 시니어 풀스택 개발자.  
* **수행 작업:** 단일 .jsx 파일 내에 상기 기술 스택과 보안 원칙을 준수하는 웹앱 구현.  
* **필수 포함 기능:** 보안 서약 모달, 메인 대시보드(차트), 프롬프트 진단 엔진 로직, 과제 관리 칸반 UI.  
* **보안 로직:** filterSensitiveData 함수 구현 필수, 하드코딩 금지 가이드 포함.

## **🚀 7\. 실행 로드맵 (Execution Roadmap)**

1. **Phase 1 (Prototype):** 보안 서약 및 과제 관리 중심의 MVP 배포.  
2. **Phase 2 (Upgrade):** AI 자가진단 엔진 고도화 및 RAG 기반 사내 헬프데스크 탑재.  
3. **Phase 3 (Identity):** 전사 ROI 대시보드 오픈 및 내부망(On-premise) 최적화 운영.

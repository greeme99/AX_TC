# **🛰️ AX Control Tower**

**중견기업의 AI 전환(AX)을 위한 통합 관리 플랫폼**

이 프로젝트는 기업 내 AI 도입의 표준화, 보안 가이드라인 준수, 교육 및 과제 관리를 통합적으로 지원하는 웹 서비스입니다.

## **🌟 주요 기능**

* **대시보드:** 전사 AI 도입 현황 및 ROI 지표 시각화  
* **보안 관리:** 실시간 프롬프트 자가진단 및 사내 보안 5대 원칙 안내  
* **템플릿 허브:** 모델별/유형별 우수 프롬프트 사례 공유 및 관리  
* **교육 센터:** 수준별(초/중/상) AI 리터러시 커리큘럼 제공  
* **과제 관리:** 4단계(아이디어-진행-검증-배포) 기반의 AX Lifecycle 관리  
* **리소스 관리:** LLM, API, AI Agent 및 인적 자원 통합 인벤토리

## **🛠️ 기술 스택**

* **Frontend:** React 18+, Tailwind CSS, Lucide React, Recharts  
* **Backend/DB:** Google Firebase (Project IDX 환경)  
* **AI Engine:** Google Gemini 2.5 Flash

## **🛡️ 보안 정책**

* 기밀 정보(개인정보, 영업비밀) 입력 방지를 위한 filterSensitiveData 로직 탑재  
* 모든 인증 정보는 환경변수(.env)를 통해 관리하며 소스코드 포함을 엄격히 금지함

*본 프로젝트는 '바이브 코더'와 함께 설계되었습니다.*
